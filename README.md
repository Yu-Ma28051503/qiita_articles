## Qiita CLI

コンテンツをブラウザでプレビューする
```
npx qiita preview
```

新しい記事を追加する
```
npx qiita new (article_name)
```

記事を投稿，更新する
```
npx qiita publish (article_name)
```

コマンドのヘルプを確認する
```
npx qiita help
```
